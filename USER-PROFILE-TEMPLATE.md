# USER-PROFILE 账号配置

> 不会填也没关系。先从下面的「小白快速版」开始，把括号里的内容改成自己的话即可。
> 如果只是先体验写文章，公众号 AppID / AppSecret 可以先留空，等需要发布到草稿箱时再补。
> `USER-PROFILE.md` 是你的私人配置，不要提交到 GitHub。

---

## 0. 小白快速版（先填这 5 行就能用）

你可以直接复制下面这段，把括号里的内容换成自己的情况：

```text
我是谁：我是一名（职业/身份），最近想开始做公众号。
我想写什么：我想写（你感兴趣或擅长的方向）。
写给谁看：主要写给（目标读者），他们常遇到的问题是（痛点）。
我想要的风格：像朋友聊天一样，真实、清楚、不要太官方。
我不想写什么：不写（不想碰的话题/风险内容/自己不懂的领域）。
```

### 示例 A：普通人副业探索

```text
我是谁：我是一名上班族，最近想尝试做公众号副业。
我想写什么：我想写普通人如何用 AI 提高工作效率、做副业、整理自己的经验。
写给谁看：主要写给下班后想做点事但不知道怎么开始的人，他们的问题是没方向、没时间、怕写不好。
我想要的风格：像朋友聊天一样，真诚、具体、少讲大道理。
我不想写什么：不写暴富、割韭菜、投资建议，也不写我没有亲自试过的方法。
```

### 示例 B：职场专业 IP

```text
我是谁：我是一名产品经理，想用公众号沉淀自己的专业经验。
我想写什么：我想写产品方法、项目复盘、AI 工具如何辅助工作。
写给谁看：主要写给 1-5 年经验的职场人，他们的问题是工作很忙但缺少系统方法。
我想要的风格：专业但不端着，逻辑清楚，有案例，有个人判断。
我不想写什么：不写空泛鸡汤、职场焦虑、未经验证的工具推荐。
```

### 示例 C：生活成长记录

```text
我是谁：我是一名宝妈/自由职业者，想重新开始写公众号。
我想写什么：我想写生活观察、个人成长、读书思考和普通人的自我恢复。
写给谁看：主要写给和我一样想重新找回自己节奏的人，他们的问题是时间碎片化、容易自我怀疑。
我想要的风格：温柔、真实、有陪伴感，不要说教。
我不想写什么：不写极端观点、不写隐私八卦、不写夸张标题党。
```
